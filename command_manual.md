# TimelineTRPG 指令手册

## 概述

TimelineTRPG 是一个基于时间线规则的 TRPG 插件，支持掷骰、检定、角色管理、战斗系统、资源管理等多种功能。

**指令前缀**: `.` `。` `#` `/` (任选其一)

## 1. 掷骰系统 (r)

### 功能
生成随机数，支持多种掷骰格式。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.r` | 掷 d100 (1-100) |
| `.r <表达式>` | 掷骰表达式，如 `3d6+2` |
| `.r <数字>` | 生成 1-该数字 之间的随机数 |
| `.r d<数字>` | 掷 1 个指定面数的骰子 |

### 示例
```
.r              # 掷 d100，结果 1-100
.r 3d6+2        # 掷 3 个 6 面骰 + 2
.r 100          # 生成 1-100 随机数
.r d20          # 掷 1 个 20 面骰
```

### 高级格式
- `N.dM`: 掷 N 个标准骰 + 小数部分比例的额外骰
  - 例如 `3.4d5.2` = 3个uniform(1, 5.2) + 1个uniform(0, 5.2)×0.4

## 2. 检定系统 (ex)

### 功能
使用角色属性进行检定，需要先创建角色。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.ex <属性名>` | 使用已设置的 tar 目标值进行检定 |
| `.ex <属性名> <目标值>` | 使用指定目标值进行检定 |
| `.ex <属性名> <数字1> <数字2>` | 使用两数字乘积作为目标值 |

### 示例
```
.ex 力量          # 使用 tar 设置的目标值进行力量检定
.ex 力量 50       # 以 50 为目标值进行力量检定
.ex 力量 10 5     # 以 10×5=50 为目标值进行力量检定
```

### 有效属性名

#### 物理属性
体质、敏捷、力量

#### 思维属性
意志、教育、智力

#### 领域属性
医学及生命科学（别名：医学）、工程与科技（别名：工程、科技）、军事与生存（别名：军事、生存）、文学、视觉及表演艺术（别名：表演、艺术）

#### 属性分组
- 物理：体质、敏捷、力量
- 思维：意志、教育、智力，以及所有领域属性
- 领域：所有领域属性

#### 英文别名
str/strength（力量）、dex/dexterity（敏捷）、con/constitution（体质）、int/intelligence（智力）、wis/wisdom（意志）、edu/education（教育）

## 3. 目标值系统 (tar)

### 功能
设置/查看检定目标值。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.tar` | 查看当前目标数值 |
| `.tar <数值>` | 直接设定目标值 |
| `.tar <等级>d` | 随机难度，例如 `5d` = 掷 5d100 取中间值 |
| `.tar <等级> <难度>` | 计算目标数值 (等级×难度÷2+50) |

### 示例
```
.tar             # 查看当前目标值
.tar 50          # 设定目标值为 50
.tar 5d          # 随机难度
.tar 5 10        # 等级5，难度10，计算目标值
```

## 4. 角色管理 (chr)

### 功能
管理多个角色，创建、选择、查看、删除角色。

### 指令格式
| 指令 | 说明 |


## 6. Buff系统 (buff)

### 功能
管理角色增益/减益效果。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.buff` | 显示当前 Buff 列表 |
| `.buff add <名称> <属性> <类型> <数值> (持续时间)` | 添加 Buff |
| `.buff show (覆盖范围)` | 查看 Buff 列表，可指定覆盖范围 |
| `.buff del <序号>/all` | 移除 Buff |

### Buff 类型
- `直接加算`: 直接加减属性值
- `直接乘算`: 按百分比乘算属性值
- `最终加算`: 在所有计算后加减属性值
- `最终乘算`: 在所有计算后按百分比乘算

### 数值格式
- 加算类型: `+20`, `-15`, `30`
- 乘算类型: `+20%`, `-10%` (会自动转换为小数)

### 示例
```
.buff add 力量Buff 力量 直接加算 5 10
.buff add 敏捷Buff 敏捷 直接乘算 +10%
.buff del 1
.buff del all
```

## 7. 资源修正 (dr)

### 功能
管理资源修正（护盾、伤害减免等）。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.dr` | 查看资源修正列表 |
| `.dr add <来源> <范围> <数值> (类型) (持续时间)` | 添加资源修正 |
| `.dr del <序号>/all` | 移除资源修正 |
| `.dr <范围>` | 显示指定范围的资源修正 |

### 参数说明
- 来源: 修正来源名称
- 范围: 生效范围 (如 hp, mp, all)
- 数值: 修正值
- 类型: (可选) 物理/魔法/自定义
- 持续时间: (可选) 持续t数

### 示例
```
.dr add 盾牌 hp 10 物理
.dr add 魔法盾 mp 15
.dr del 1
.dr hp
```

## 8. 资源记录 (rc)

### 功能
管理角色的 HP、MP、护盾等资源。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.rc` | 显示资源状态 |
| `.rc <hp/mp> <变化值> (持续时间) (f)` | 修改 HP/MP |
| `.rc 护盾/s <变化值> (覆盖范围) (持续时间) (f)` | 添加护盾 |
| `.rc reset` | 重置资源到最大值 |
| `.rc show` | 显示当前资源状态 |

### 参数说明
- `变化值`: `+10` 增加，`-10` 减少
- `f` 标记:  (可选) 允许超过上限
- 持续时间:  (可选) 持续t数

### 示例
```
.rc hp +20         # HP 增加 20
.rc hp -10 f       # HP 减少 10，允许超过下限
.rc mp +30 5       # MP 增加 30，持续 5 回合
.rc s 10           # 添加 10 点护盾
.rc s 5 物理        # 添加 5 点物理护盾
.rc reset          # 重置资源
.rc show           # 显示资源状态
```

## 9. 背包管理 (i)

### 功能
管理角色的物品和现金。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.i` | 显示背包内容和负重状态 |

## 14. 时间线管理 (tl)

### 功能
管理战斗时间线。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.tl` | 显示时间线列表 |
| `.tl new <名称>` | 创建新时间线 |
| `.tl <序号>` | 选择时间线 |
| `.tl del <序号>/all` | 删除时间线 |
| `.tl show` | 显示时间线内容 |
| `.tl get <时间点>` | 查询指定时间点的行动 |
| `.tl acc <精度>` | 设置时间精度 |

### 时间精度
- `0`: 整数 (1, 2, 3...)
- `1`: 1 位小数 (0.1, 0.2...)
- `2`: 2 位小数 (0.01, 0.02...)
- 以此类推

### 示例
```
.tl                 # 显示时间线列表
.tl new 第一次战斗  # 创建名为"第一次战斗"的时间线
.tl 1               # 选择第 1 个时间线
.tl show            # 显示当前时间线内容
.tl get 5           # 查询 5t 时的行动
.tl acc 2           # 设置时间精度为 0.01
```

## 15. 战斗系统 (bt)

### 功能
时间线战斗系统核心模块。

### 指令格式
| 指令 | 说明 |
|------|------|
| `.bt <属性> <时间>t (笔记)` | 添加战斗行动 (时间模式) |
| `.bt <属性> <影响值> (笔记)` | 添加战斗行动 (影响值模式) |
| `.bt wp <时间>t/<影响值> (笔记)` | 使用武器进行战斗 |
| `.bt undo` | 撤销最后行动 |
| `.bt <时间t> <属性> <时间>t/<影响值>` | 在指定时间插入行动 |
| `.bt wp re` | 装填火力武器 |

### 时间模式 vs 影响值模式
- **时间模式**: `.bt 力量 5t` → 根据时间计算影响值
- **影响值模式**: `.bt 力量 50` → 根据影响值计算时间

### 统一公式
```
影响值 = 属性值 × 时间 / 20
时间 = 影响值 × 20 / 属性值
```

### 武器战斗说明
- **增幅武器**: `.bt wp 5t` 或 `.bt wp 50` 都可以
- **火力武器**: `.bt wp 50` (影响值模式)

### 示例
```
.bt 力量 5t         # 力量 5t 时间，属性通过buff增幅后计算影响值
.bt 敏捷 30         # 敏捷属性，影响值30
.bt wp 10t         # 使用武器，10t时间
.bt wp 50          # 使用武器，影响值50
.bt 3t 力量 5t     # 在3t时插入行动
.bt undo           # 撤销最后行动
.bt wp re          # 装填火力武器
```

### 数据存储
- 角色数据: `plugin_data/TimelineTRPG/User/{user_id}/characters.json`
- 武器数据: `plugin_data/TimelineTRPG/User/{user_id}/weapons.json`
- 背包数据: `plugin_data/TimelineTRPG/Inventory/{group_id}/items.json`
- 战斗数据: `plugin_data/TimelineTRPG/Battle/{group_id}/timeline.json`
- 目标值: `plugin_data/TimelineTRPG/Examination/target.json`

2. 战斗系统需要先创建时间线 (`.tl new`)
3. 武器战斗需要先装备武器 (`.wp 序号`)
4. 火力武器需要装填弹药 (`.bt wp re`)
